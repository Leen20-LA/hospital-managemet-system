# Hospital Managemet System

## Description
A Java-based console application for managing hospital surgeries, doctor appointments, and patient bookings. The system allows users to view surgeons, doctors, appointments, make bookings, and save/load data from text files.It also includes a simple GUI for patient registration.

(This project was developed as a team project for a university course.)

## Technologies Used
 - Java
 - Opject-Oriented Programming(OOP)
 - JavaFX for GUI
 - NetBeans IDE
    
## Features
 - View all surgeons – Display a list of all surgeons with their specialties and doctors.
 - Make a booking – Choose a surgeon, doctor, and available appointment slot to book for a patient.
 - Check current booking – View the details of your most recent booking.
 - Save and load data – Store surgeon information in a text file and read it back when needed.
